Java加强编译运行

加强compile&runingtime

  • 不要用有参构造函数
  • 使用移位操作代替乘除法

    a/4 == a >> 2
    a/8 == a >> 3
    a×4 == a << 2
    a×8 == a << 3

  • 类上申明final
  • for使用如下方式

    for(int i=0,length = list.length; )

减少内存开销

  • 批量创建对象是使用clone方法
  • 尽量使用局部变量
